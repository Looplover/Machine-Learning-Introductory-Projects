### 🚀 Projects Overview

</div>

#### 🟢 Regression & Discriminant Analysis
Built **Batch and Stochastic Gradient Descent (SGD)** for Linear Regression, analyzing error surfaces and comparing mini-batch efficiencies against closed-form mathematical solutions. Implemented Logistic Regression iteratively optimized via **Newton's Method**. Designed **Gaussian Discriminant Analysis (GDA)** models, visualizing the bias-variance tradeoff between linear (shared covariance) and quadratic (distinct covariance) decision boundaries.

#### 🟣 Text Classification & Support Vector Machines
Developed a **Naive Bayes** classifier for text data, examining the exact impact of n-grams (unigrams vs. bigrams), stemming, and stopword removal. Implemented soft-margin **Support Vector Machines (SVM)** from scratch for image classification by mapping and solving the dual optimization problem via `cvxopt`. Designed One-vs-One multiclass SVMs utilizing both Linear and Gaussian kernels, systematically benchmarking them against `sklearn`.

#### 🔵 Decision Trees, Random Forests & Neural Networks
Engineered **Decision Trees** capable of handling one-hot encoded categorical features and applied rigorous post-pruning techniques (Entropy and Gini impurity) to combat severe overfitting. Scaled models to **Random Forest** ensembles for significantly improved generalization. Built **Deep Neural Networks**, analyzing hidden-layer capacity, early stopping protocols, and the vanishing gradient problem by comparing Sigmoid vs. ReLU activations across varying network depths.

#### 🟢 Seq2Seq Spatial Reasoning (Maze Pathfinding)
Designed sequence-to-sequence (Seq2Seq) architectures to autonomously navigate and solve spatial mazes. By feeding tokenized adjacency list representations into a **Recurrent Neural Network (RNN) with Bahdanau Attention** and a modern **Transformer-based** model, the system learned to accurately predict and generate optimal coordinate paths from start to finish.
